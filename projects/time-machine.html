Time Machine

Description

With this time machine, I'll be able to go back into the past and stop myself from cutting my own hair.

But then... I wouldn't NEED to make the time machine... soooo, I wouldn't have gone back in time... sooo I WOULD need a time-machine... so... whatever! Let's DO THE DAMN THING.

Materials

  • One large refrigerator box (the base station)
  • Two medium-sized plastic tubs (for temporal engines)
  • Aluminum foil (to cover and style engines)
  • Colored buttons and switches (for the control panel)
  • LED lights and batteries (to light up the control panel)
  • Wiring and electrical tape (to connect the LEDs)
  • Metallic spray paint (for a futuristic finish)
  • Permanent markers (for designing and detailing)
  • A big fan (for rear propulsion... or just looks)
  • Scissors, glue, duct tape

Steps

  1. Cut and arrange the refrigerator box to create the main structure of the time machine.
  2. Attach plastic tubs to the sides as temporal engines using duct tape.
  3. Cover the tubs with aluminum foil and accentuate with metallic spray paint.
  4. Create a control panel using cardboard, affixing buttons and switches.
  5. Install LED lights behind the control panel for functional lighting effects.
  6. Draw circuits with permanent markers and set up wiring for the LEDs.
  7. Use the fan as a rear propeller, decorating it to match the time machine.
  8. Ensure all connections are secure with electrical tape and check the wiring.
  9. Decorate the exterior with markers to add details such as time circuits.
  10. Perform a final inspection to ensure everything is operational and safe.
